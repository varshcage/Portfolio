# Varshaan Portfolio

A portfolio site built with React, TypeScript, Tailwind CSS, and PrimeReact.

## Getting Started

### Prerequisites

- Node.js 20, 22, or 24
- npm

### Install

```bash
npm install
```

### Run the dev server

```bash
npm run dev
```

Then open `http://localhost:5173`.

### Build for production

```bash
npm run build
```

The production output is written to `dist/`.
